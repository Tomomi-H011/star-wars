# About App

This app is built with React Native and uses the Star Wars API ([swapi.tech](https://www.swapi.tech/)) to display planets, films, and spaceships data.

## How to Use

- Slide the buttons to the left to display details of planets, films, and spaceships.
- Use the search box to filter the data.
